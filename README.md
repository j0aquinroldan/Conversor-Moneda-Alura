# Conversor de Moneda

Este es un conversor de monedas que por el momento funciona con las siguientes divisas:

- `Peso Argentino`
- `Euro`
- `Dolar`
- `Libra Esterlina`
- `Yen Japones`
- `Won Sul-Coreano` 

# En este primer caso, se convierte de euro a peso argentino.
